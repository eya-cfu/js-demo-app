Demo fullstack js contact's list webapp.
Ref tuto : https://buddy.works/tutorials/build-a-full-stack-web-application-with-angular-7-and-nest-js
Frontend app with angular, backend REST server with nest js.

App uses a MySQL 8.0 server. Please start MySQL server and load the database under /db.

Backend uses Node JS 12.14.1. To start server, run:
$ npm install
$ npm run start:dev
Server is at localhost:3000
Frontend uses Angular 11.2.0. To start server, run:
$ npm install
$ ng serve
Then open browser on localhost:4200
